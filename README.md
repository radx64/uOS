# uOS
micro Operating System for x86 architecture

Build status: [![Build Status](https://travis-ci.org/radx64/uOS.svg)](https://travis-ci.org/radx64/uOS)


Functionalities:
- Bootable
- Knows how to handle Frame Buffer (kind of)
- Text cursor support
